Predicting NBA All Star Teams for the 2020-2021 NBA Seasons

In this notebook we will provide an analysis of how likely a player is to make an NBA team using general in game statistics per game. We will use a Logistic Regression to predict a percentage of making an all star team based on the following statistics provided at https://www.basketball-reference.com/leagues/NBA_2021_per_game.html. Note that I had to manually add in a denoting if a player made an allstar game or not. This model is descriptive in telling us what effects each statistic has on making the all star team but also can be predictive in determining who will make the all star game in this season. There are also interesting insights such as who just missed the team and who probably should have made the team.

def clean_name(string):
    stripped = string.split("\\", 1)[0]
    return stripped
df["Player"] = df["Player"].apply(clean_name)
#marks all duplicates as true
mask = ~df["Player"].duplicated(keep = False)
df = df[(mask) | (df["Tm"] == "TOT")]

Conclusion: We found good predictors to predict which players will make the all star team.

We found variables we needed to remove that wre not good predictors. Some were surprising as Assists is a pretty important statistic.

Some problems I ran into were players who did not shoot 3 point shots leaving blanks in the data set. To fix this we removed them as all of the all stars had all stats filled out.

Suggestions for future research would be to utilize a bigger data set with some recent years. We could not go back too far as the parameters fro making the team have changed. Additionally, we could try different variables that could also have predictive power. We could also perform a similar analysis for other nba awards and see the correlation with this model.
